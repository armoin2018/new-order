    Class AiPerceptionEngine

    Stateless engine that evaluates AI perception of the player-leader and provides expanded 8-faction AI personality profiles.

    Every public method is pure: no mutation of this, no side effects, all results are freshly-allocated objects.

    computePerceivedProfile

    • computePerceivedProfile(
          actualProfile: LeaderPsychology,
          accuracy: number,
      ): PerceivedProfileResult

      Given the actual player psychology and an AI faction's perception accuracy, compute what the AI "thinks" the player's profile looks like.

      At 100% accuracy the perceived profile equals the actual profile. At lower accuracy, numeric dimensions drift toward 50 (neutral).

      Drift formula: perceived = actual + (50 − actual) × (1 − accuracy / 100)

      Non-numeric dimensions (decisionStyle, stressResponse) are preserved as-is — the AI always knows the player's decision style.

      Parameters

      • actualProfile: LeaderPsychology

        The player's real psychological profile.

      • accuracy: number

        The AI faction's current perception accuracy (0–100).

      Returns PerceivedProfileResult

      A PerceivedProfileResult containing the perceived profile and magnitude of drift.

    computePerceptionDrift

    • computePerceptionDrift(
          currentState: AiPerceptionState,
          turnsElapsed: number,
      ): PerceptionDriftSummary

      Summarise how much each AI faction's perception of the player has drifted from the initial accuracy baseline.

      Drift is measured as the absolute difference between the initial accuracy and the current accuracy for each faction. Positive drift indicates the AI's model has degraded.

      A faction is considered "critically low" if its accuracy is below 30.

      Parameters

      • currentState: AiPerceptionState

        Current perception state with per-faction accuracies.

      • turnsElapsed: number

        Number of turns since perception tracking began.

      A PerceptionDriftSummary with per-faction and aggregate metrics.

    evaluateDesperationMode

    • evaluateDesperationMode(
          powerBase: PowerBase,
          stability: number,
          civilUnrest: number,
      ): DesperationModeResult

      Determine whether an AI leader has entered Desperation Mode.

      Desperation Mode activates when any of these conditions are true:

      • stability < 20
      • any PowerBase sub-score < 15
      • civilUnrest > 80

      While in Desperation Mode the leader's psychology is additively modified: riskTolerance +30, paranoia +20, patience −30.

      Parameters

      • powerBase: PowerBase

        The leader's current domestic power base.

      • stability: number

        Current national stability (0–100).

      • civilUnrest: number

        Current civil unrest level (0–100).

      Returns DesperationModeResult

      A DesperationModeResult with trigger info and modifiers.
