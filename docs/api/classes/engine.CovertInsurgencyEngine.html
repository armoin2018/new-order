    Class CovertInsurgencyEngine

    Manages the lifecycle of covert insurgency operations: validation, initiation, per-turn effect processing, and cancellation.

    All public methods are pure with respect to external game state — they return delta objects rather than mutating NationState directly. The caller (turn engine) is responsible for applying the deltas.

    • canInitiateInsurgency(
          sponsor: NationState,
          activeCount: number,
      ): InsurgencyValidation

      Determine whether a sponsoring nation meets all preconditions to initiate a new covert insurgency.

      Checks (in order):

      1. Diplomatic Influence ≥ InsurgencyConfig.minDIToInitiate
      2. Treasury ≥ InsurgencyConfig.treasuryCostPerTurn
      3. Active insurgency count < InsurgencyConfig.maxConcurrentInsurgencies

      Parameters

        Current nation state of the would-be sponsor.

      • activeCount: number

        Number of insurgencies already funded by this sponsor.

    processTurnEffects

    • processTurnEffects(
          insurgency: InsurgencyRecord,
          currentTurn: TurnNumber,
      ): InsurgencyTurnResult

      Compute the effects of a single insurgency for the current turn.

      • During the ramp-up period (currentTurn < effectiveTurn): returns zero deltas with inRampUp: true.
      • Once effective: returns stabilityDelta = -stabilityReductionPerTurn and treasuryCost = treasuryCostPerTurn.

      FR-401 invariant: no war-state is triggered — the returned deltas never include a war-state flag.

      Parameters

      • insurgency: InsurgencyRecord

        The insurgency record to process.

      • currentTurn: TurnNumber

        The current game turn.

      Returns InsurgencyTurnResult

      Deltas for this turn (stability hit + treasury cost).
