    Class PsychologicalJourneyEngine

    Post-game Psychological Journey analysis engine.

    Produces all data structures needed to render the four-panel psychological journey screen (emotional chart, radar chart, grudge timeline, and turning points). All methods are pure — no side-effects, no mutation.

    detectPsychologicalTurningPoints

    • detectPsychologicalTurningPoints(
          params: {
              emotionalTimeline: readonly EmotionalTimelineEntry[];
              leaderId: LeaderId;
              threshold?: number;
          },
      ): TurningPointResult

      Algorithmically detect psychological turning points in an emotional timeline.

      Algorithm: For each consecutive pair of turns, check each emotional dimension (stress, confidence, anger, fear, resolve). If the new value ≥ psychologicalTurningPointThreshold and |delta| ≥ psychologicalShiftMinDelta, the pair is recorded as a turning point.

      Parameters

      • params: {
            emotionalTimeline: readonly EmotionalTimelineEntry[];
            leaderId: LeaderId;
            threshold?: number;
        }
        • ReadonlyemotionalTimeline: readonly EmotionalTimelineEntry[]

          — Ordered timeline entries from buildEmotionalTimeline.

        • ReadonlyleaderId: LeaderId

          — The leader to analyse.

        • Optional Readonlythreshold?: number

          — Optional override for the emotion threshold (defaults to config value).

      Returns TurningPointResult

      A TurningPointResult with all detected turning points.
