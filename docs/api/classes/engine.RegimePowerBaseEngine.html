    Class RegimePowerBaseEngine

    Stateless engine that evaluates martial law, regime change, power base erosion, and coup attempt mechanics.

    Every public method is pure: no mutation of this, no side effects, all results are freshly-allocated objects.

    • applyPowerBaseErosion(
          currentPowerBase: PowerBase,
          actionCategory: string,
      ): PowerBaseErosionResult

      Apply erosion deltas to a power base for a given action category.

      Each action category (e.g. unpopularWar, economicSanctions) has a predefined delta table. All resulting values are clamped to [0, 100].

      If the action category is not recognised, no deltas are applied and all deltas in the result are zero.

      Parameters

      • currentPowerBase: PowerBase

        The power base before erosion.

      • actionCategory: string

        Key into the erosionPerAction config table.

    evaluateCoupAttempt

    • evaluateCoupAttempt(powerBase: PowerBase, popularity: number): CoupAttemptResult

      Evaluate whether a coup attempt occurs and determine its outcome.

      Eligibility: PowerBase.military < 30 AND PowerBase.securityServices < 30.

      Formula (FR-1311):

      CoupChance = (100 − military) × 0.3
                 + (100 − securityServices) × 0.3
                 − Popularity × 0.2

      Outcome determination (deterministic, based on coupChance):

      • coupChance ≥ 60 → Success
      • coupChance ≥ 30 → Failed (attempted but repelled)
      • coupChance < 30 → Averted (plot discovered before execution)

      Parameters

      • powerBase: PowerBase

        Current power base scores.

      • popularity: number

        Leader popularity (0–100).

      Returns CoupAttemptResult

      A CoupAttemptResult with eligibility, chance, and outcome.

    executeRegimeChange

    • executeRegimeChange(
          currentPsychology: LeaderPsychology,
          currentPowerBase: PowerBase,
          isPlayer: boolean,
          successionClarity: number,
      ): RegimeChangeResult

      Execute a regime change on a faction.

      AI factions: new leader inherits a randomized personality (±20 on numeric psychology dimensions) and a reduced power base.

      Player faction: Game Over unless SuccessionClarity > 60, in which case the player can continue as a successor with a damaged state.

      Psychology randomization uses a deterministic pattern: for each numeric key at index i, delta = personalityRandomizationRange × ((i % 2 === 0) ? 1 : -1) × ((i + 1) / 6).

      Parameters

      • currentPsychology: LeaderPsychology

        The outgoing leader's psychological profile.

      • currentPowerBase: PowerBase

        The outgoing leader's power base.

      • isPlayer: boolean

        Whether this faction is controlled by the player.

      • successionClarity: number

        SuccessionClarity value from LeaderVulnerabilities.

      Returns RegimeChangeResult

      A RegimeChangeResult describing the outcome.
