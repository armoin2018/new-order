    checkVulnerabilityTrigger

    • checkVulnerabilityTrigger(
          vulnerabilityType: string,
          currentTurn: TurnNumber,
          triggersFired: number,
          deterministicRoll: number,
      ): VulnerabilityTriggerResult

      Determine whether a personal vulnerability event should fire this turn.

      Preconditions (all must be true for a trigger):

      • currentTurn ≥ config.vulnerabilityEvents.earliestTriggerTurn
      • triggersFired < config.vulnerabilityEvents.maxTriggersPerGame
      • The vulnerability uses random per-turn triggering (IdeologicalRigidity does NOT — it fires on inconsistent actions, not randomly).
      • deterministicRoll (0–1) < configured triggerChancePerTurn.

      Parameters

      • vulnerabilityType: string

        The vulnerability key.

      • currentTurn: TurnNumber

        Current game turn.

      • triggersFired: number

        How many times this vulnerability has already fired.

      • deterministicRoll: number

        A pre-generated random value in [0, 1).

      Returns VulnerabilityTriggerResult

      Whether the event triggers and why.

    computeArchetypeDeviation

    • computeArchetypeDeviation(
          psychology: LeaderPsychology,
          archetype: string,
      ): ArchetypeDeviationResult

      Calculate how far the current psychology has deviated from an archetype preset.

      For each numeric dimension, the signed delta is current - archetypePreset. The totalDeviation is the sum of the absolute values of all per-dimension deltas. A high total deviation means the player has substantially customised away from the archetype.

      If the archetype key is invalid, all deltas are 0 and total deviation is 0.

      Parameters

      • psychology: LeaderPsychology

        The leader's current psychological profile.

      • archetype: string

        The archetype key to compare against.

      Returns ArchetypeDeviationResult

      Per-dimension deltas and aggregate deviation score.

    computeOngoingVulnerabilityEffect

    • computeOngoingVulnerabilityEffect(
          vulnerabilityType: string,
          turnsSinceTriggered: number,
      ): OngoingVulnerabilityResult

      For duration-based vulnerabilities (HealthRisk, SuccessionGap), check whether the effect is still active and compute per-turn effects.

      • HealthRisk: Active for durationTurns (2). Leader is incapacitated while active.
      • SuccessionGap: Active for durationTurns (5). Stability penalty of −5 per turn while active.
      • All other types return an inactive result (no ongoing component).

      Parameters

      • vulnerabilityType: string

        The vulnerability key.

      • turnsSinceTriggered: number

        Number of turns elapsed since the event fired.

      Returns OngoingVulnerabilityResult

      Whether the effect is still active and its per-turn impact.

    computeSliderPreview

    • computeSliderPreview(
          archetype: string | null,
          overrides: Partial<LeaderPsychology>,
      ): SliderPreviewResult

      Compute a live preview of the leader's psychology given an optional archetype and player overrides.

      Resolution order:

      1. If archetype is provided and valid, start from that preset. Otherwise start from neutral defaults (50 for numerics, 'Analytical' / 'Consolidate' for enums).
      2. Apply overrides on top.
      3. Clamp all numeric values to the configured slider range.

      Parameters

      • archetype: string | null

        Archetype key or null for neutral base.

      • overrides: Partial<LeaderPsychology>

        Partial psychology overrides from the player.

      Returns SliderPreviewResult

      The preview result with final psychology and metadata.

    evaluateLeaderBalance

    • evaluateLeaderBalance(psychology: LeaderPsychology): LeaderBalanceAssessment

      Assess whether a leader's psychology is balanced or extreme.

      Examines all 6 numeric dimensions. A dimension is "extreme" if its value falls in [0, 15] or [85, 100]. Rating scale:

      • 0 extremes → 'Balanced'
      • 1–2 extremes → 'Moderate'
      • 3–4 extremes → 'Extreme'
      • 5–6 extremes → 'Radical'

      Also computes the average and standard deviation of all numeric dimension values.

      Parameters

      • psychology: LeaderPsychology

        The leader's current psychological profile.

      Returns LeaderBalanceAssessment

      Balance assessment with statistics.
