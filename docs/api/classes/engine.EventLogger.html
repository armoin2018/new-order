    Class EventLogger

    Append-only event logger for the simulation engine.

    Records every game event with an auto-generated unique ID and ISO-8601 timestamp. Supports deterministic ID generation when constructed with a SeededRandom instance, ensuring replay-safe identifiers.

    Usage

    const logger = new EventLogger(rng);
    logger.logEvent({ turn: 1 as TurnNumber, category: EventCategory.Military, ... });
    const events = logger.getByTurn(1 as TurnNumber);
