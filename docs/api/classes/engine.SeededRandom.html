    Class SeededRandom

    A seedable pseudo-random number generator using the mulberry32 algorithm.

    Usage

    const rng = new SeededRandom(42);
    const roll = rng.nextInt(1, 6); // deterministic d6

    Determinism guarantee

    Given the same seed and the same sequence of method calls, the output is identical across runs, platforms, and JS engines.

    next

    • next(): number

      Return the next pseudo-random number in [0, 1) — a deterministic drop-in replacement for the native JS random function.

      Uses the mulberry32 algorithm:

      • Period: 2³² (full-period)
      • Quality: passes gjrand and PractRand up to 2³² outputs
      • Speed: ~1 ns per call in modern engines

      Returns number

      A float in the half-open interval [0, 1).
