    Class SaveManager

    Manages explicit named save slots in localStorage with full checksum validation.

    Design notes

    • Never throws — all recoverable errors are returned via result objects.
    • React-free: safe for use on the main thread or inside a Web Worker.
    • Separate from Zustand's persist auto-save (conflict-2026-save key).
