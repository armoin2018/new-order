    Class RepressionEngine

    Pure-function engine for computing repression outcomes, backlash feedback, martial law effects, and coup risk assessments.

    All public methods are side-effect-free and return new result objects. The engine is parameterised by RepressionConfig to allow scenario-specific tuning without code changes.

    assessCoupRisk

    • assessCoupRisk(powerBase: PowerBase, popularity: number): CoupRiskAssessment

      Assess the probability of a coup attempt based on the leader's domestic power base and popular approval.

      A coup is only possible when both powerBase.military and powerBase.securityServices fall below their configured thresholds. When possible, the probability is computed as:

      CoupChance = (100 − military) × militaryWeight
                 + (100 − securityServices) × securityServicesWeight
                 − popularity × popularityWeight

      The result is clamped to [0, 100].

      Parameters

      • powerBase: PowerBase

        The leader's domestic power base loyalty scores.

      • popularity: number

        The leader's current popular approval (0–100).

      Returns CoupRiskAssessment

      A CoupRiskAssessment with probability and explanation.

      See

      FR-1311 — Coup attempt probability formula

      • Defined in engine/repression-engine.ts:294

    computeBacklashFeedback

    • computeBacklashFeedback(currentBacklash: number): BacklashFeedback

      Compute the backlash-to-unrest feedback contribution.

      When repression backlash exceeds the configured threshold, the excess feeds back into civil unrest at a rate of 0.5 per point above the threshold — modelling the "Dictator's Trap" where heavy-handed repression inflames the population.

      Parameters

      • currentBacklash: number

        Current repression backlash level (0–100).

      Returns BacklashFeedback

      A BacklashFeedback describing whether the loop is active and its unrest contribution.

      See

      FR-1304 — Backlash feedback: unrest += (backlash − threshold) × 0.5
