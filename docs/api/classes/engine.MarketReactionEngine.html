    Class MarketReactionEngine

    Scans headlines for pre-configured trigger keywords and computes the resulting GlobalInflation delta each turn.

    Stateless — all mutable context (cooldowns, inflation) is passed in and returned rather than stored, making the engine trivially testable and compatible with the turn-engine pipeline.
