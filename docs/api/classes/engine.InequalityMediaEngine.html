    • computeInequality(input: InequalityInput): InequalityResult

      Compute the new inequality score for a nation this turn.

    computeSuccessProbability

    • computeSuccessProbability(pressFreedom: number): number

      Compute the success probability for a media-manipulation attempt given a nation's press-freedom index.

      Formula:

      P = baseSuccessProbability + (100 − pressFreedom) × pressFreedomModifier

      Lower press freedom → higher success probability (easier to suppress media in autocracies).

      Parameters

      • pressFreedom: number

        Press-freedom index (0–100).

      Returns number

      Clamped probability in [0, 1].
