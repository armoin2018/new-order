    Interface ViralityDeepfakeConfig

    Configuration subset consumed by ViralityDeepfakeEngine.

    Mirrors the relevant portions of GAME_CONFIG.infoWar so the engine can be instantiated with test-friendly overrides without pulling the full game config.
