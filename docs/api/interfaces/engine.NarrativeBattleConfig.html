    Interface NarrativeBattleConfig

    Configuration values consumed by the NarrativeBattleEngine.

    Mirrors the shape of GAME_CONFIG.infoWar.narrativeBattle, GAME_CONFIG.infoWar.whistleblower, and GAME_CONFIG.infoWar.whistleblowerCascade so that callers can supply test overrides without importing the full game config.
