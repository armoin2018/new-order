    Interface ClarityResult

    Per-rival Fog of War clarity result.

    Clarity = (OwnIntelCapability + AllyIntelSharing) − TargetCounterIntel, clamped to [0, 100]. Low clarity triggers ghost units and degrades headline reliability.
